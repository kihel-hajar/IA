# Présentation du Projet

Ce projet s’inscrit dans le cadre du module **Intelligence Artificielle** et vise à appliquer les techniques de **Machine Learning supervisé** afin de résoudre un problème de classification.

L’objectif principal est de **mettre en œuvre plusieurs modèles de classification avec la bibliothèque Python Scikit-learn**, d’évaluer leurs performances et de comparer leurs résultats afin d’identifier le modèle le plus performant.

Le travail comprend plusieurs étapes essentielles du processus de Machine Learning, notamment :

* la préparation et le nettoyage des données
* la division du dataset en données d’entraînement et de test
* l’entraînement de plusieurs modèles de classification
* l’évaluation des performances à l’aide de différentes métriques

---

# Objectif du Notebook

Le notebook a pour objectif pédagogique de :

* comprendre le fonctionnement du **Machine Learning supervisé**
* implémenter plusieurs **algorithmes de classification**
* comparer leurs performances sur un même dataset
* interpréter les résultats obtenus.

Les modèles testés dans ce projet sont :

* **Logistic Regression**
* **K-Nearest Neighbors (KNN)**
* **Support Vector Machine (SVM)**
* **Decision Tree**
* **Random Forest**

Le dataset utilisé est **Breast Cancer Dataset**, disponible dans la bibliothèque **Scikit-learn**, qui permet de réaliser une **classification binaire** entre tumeur bénigne et tumeur maligne.
